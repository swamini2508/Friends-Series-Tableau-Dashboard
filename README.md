# Friends Series Dashboard 📺

## Project Overview
This Tableau dashboard provides an interactive analysis of the TV series *Friends*. It explores episode ratings, seasons, characters, and other insights through interactive visualizations.

## Tools Used
- Tableau
- CSV Dataset

## Dashboard Features
- Episode ratings analysis
- Season-wise insights
- Character analysis
- Interactive filters
- Data visualizations
